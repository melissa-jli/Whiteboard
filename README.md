#Whiteboard

Whiteboard is a webapp build to provide an easy interface to collaborate on drawings and diagrams in real-time with minimal to no initial setup required.

##To run

###Install dependencies
`npm install`

###Run mongodb
On Mac OS: `mongod`
On Linux: `mongodb`

###Run app
`npm start`
